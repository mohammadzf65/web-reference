مقدمه و ساختار کلی HTML5

هر سند HTML با یک ساختار پایه شروع می‌شود که به مرورگر می‌گوید چگونه محتوای صفحه را تفسیر کند.

مشاهده کد و توضیحات

ساختار پایه HTML


<!DOCTYPE html>
<html lang="fa" dir="rtl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>عنوان صفحه من</title>
    <!-- اینجا لینک به فایل‌های CSS و سایر متادیتا قرار می‌گیرد -->
    <link rel="stylesheet" href="css/styles.css">
</head>
<body>
    <!-- محتوای اصلی صفحه اینجا قرار می‌گیرد -->
    <header>
        <h1>عنوان اصلی وب‌سایت</h1>
    </header>

    <main>
        <section>
            <h2>بخش اول</h2>
            <p>این یک پاراگراف در بخش اول است.</p>
        </section>
    </main>

    <footer>
        <p>&copy; 2023 نام وب‌سایت. تمامی حقوق محفوظ است.</p>
    </footer>
</body>
</html>

  • **<!DOCTYPE html>**: اعلامیه HTML5.
  • **<html lang="fa" dir="rtl">**: تگ ریشه سند با تنظیم زبان فارسی و جهت راست به چپ.
  • **<head>**: شامل متادیتا (مثل عنوان، کدگذاری، ریسپانسیو و لینک CSS).
  • **<body>**: حاوی تمام محتوای قابل مشاهده صفحه.

تگ‌های ساختاری و معنایی (Semantic Tags)

HTML5 تگ‌های جدیدی را معرفی کرده که به بخش‌های مختلف صفحه معنای خاصی می‌دهند.

<header>

<header>

معمولاً شامل محتوای مقدماتی یا ابزارهای ناوبری.


<header>
    <img src="logo.png" alt="لوگو وب‌سایت">
    <nav>
        <ul>
            <li><a href="#">صفحه اصلی</a></li>
            <li><a href="#">درباره ما</a></li>
        </ul>
    </nav>
</header>
<nav>

<nav>

برای گروه‌بندی لینک‌های ناوبری اصلی.


<nav>
    <ul>
        <li><a href="#about">درباره ما</a></li>
        <li><a href="#services">خدمات</a></li>
    </ul>
</nav>
<main>

<main>

محتوای اصلی و منحصر به فرد سند. فقط یک تگ <main> در هر صفحه.


<main>
    <article>
        <h1>عنوان مقاله</h1>
        <p>محتوای مقاله...</p>
    </article>
</main>
<section>

<section>

یک بخش مستقل از سند، معمولاً با یک عنوان.


<section>
    <h2>خدمات ما</h2>
    <p>ما خدمات متنوعی را ارائه می‌دهیم...</p>
</section>
<article>

<article>

محتوای مستقل و خودکفا (مثل پست وبلاگ).


<article>
    <h3>پست وبلاگ: نکاتی برای کدنویسی بهتر</h3>
    <p>اینجا محتوای کامل پست وبلاگ قرار می‌گیرد.</p>
</article>
<aside>

<aside>

محتوای جانبی یا حاشیه‌ای (مثل سایدبار).


<aside>
    <h4>تبلیغات</h4>
    <p>اینجا فضای تبلیغات است.</p>
</aside>
<footer>

<footer>

اطلاعات پایین صفحه (کپی‌رایت، تماس).


<footer>
    <p>&copy; 2023 وب‌سایت من</p>
    <a href="#">سیاست حریم خصوصی</a>
</footer>
<div>

<div>

تقسیم‌بندی عمومی بدون معنای خاص (برای گروه‌بندی با CSS/JS).


<div class="card-container">
    <div class="card">
        <h3>محصول ۱</h3>
    </div>
</div>

تگ‌های متنی اساسی

<h1> تا <h6>

<h1> تا <h6>

تگ‌های عنوان (Heading) برای سلسله‌مراتب محتوا.


<h1>عنوان اصلی مقاله</h1>
<h2>زیرعنوان بخش اول</h2>
<h3>زیرزیرعنوان</h3>
<p>

<p>

برای تعریف یک پاراگراف از متن.


<p>این یک پاراگراف ساده از متن است.</p>
<a>

<a>

برای ایجاد یک لینک (hyperlink).


<a href="https://www.google.com" target="_blank" title="رفتن به گوگل">بازدید از گوگل</a>
<img>

<img>

برای قرار دادن یک تصویر.


<img src="images/my-image.jpg" alt="توضیح مربوط به تصویر" width="300" height="200">
<ul>, <ol>, <li>

<ul>, <ol>, <li>

لیست‌های نامرتب (Unordered) و مرتب (Ordered) با آیتم‌های لیست.


<ul>
    <li>قهوه</li>
    <li>کتاب</li>
</ul>

<ol>
    <li>برنامه‌ریزی</li>
    <li>اجرا</li>
</ol>
<span>

<span>

یک کانتینر خطی (inline) برای استایل‌دهی به بخشی از متن.


<p>این یک <span style="color: blue;">متن آبی</span> است.</p>
<strong>, <em>

<strong>, <em>

<strong> برای متن مهم (بولد) و <em> برای تأکید (ایتالیک).


<p>این یک پیام <strong>بسیار مهم</strong> است.</p>
<p>من واقعا <em>این کتاب</em> را دوست دارم.</p>
<br>, <hr>

<br>, <hr>

<br> برای شکست خط اجباری و <hr> برای خط افقی جداکننده.


آدرس من:<br>
خیابان اول، پلاک ۱۰<br>
<hr>
<p>شروع بخش دوم.</p>

تگ‌های مربوط به فرم‌ها (Forms)

فرم‌ها برای جمع‌آوری اطلاعات از کاربران استفاده می‌شوند.

<form>

<form>

تگ اصلی برای ایجاد یک فرم HTML.


<form action="/submit-form" method="POST">
    <!-- عناصر ورودی -->
</form>
<input> (انواع مختلف)

<input>

پرکاربردترین تگ برای ایجاد انواع ورودی‌ها (text, password, email, checkbox, radio, etc.).


<label for="username">نام کاربری:</label>
<input type="text" id="username" name="username" placeholder="نام خود را وارد کنید" required>
<br>
<label for="password">رمز عبور:</label>
<input type="password" id="password" name="password" required>
<br>
<input type="radio" id="male" name="gender" value="male">
<label for="male">مرد</label>
<input type="checkbox" id="coding" name="interests" value="coding">
<label for="coding">برنامه‌نویسی</label>
<br>
<input type="submit" value="ارسال">
<label>, <textarea>, <select>, <option>, <button>

<label>, <textarea>, <select>, <option>, <button>

عناصر دیگر فرم برای برچسب‌گذاری، متن چندخطی، منوهای کشویی و دکمه‌ها.


<label for="comments">نظرات شما:</label>
<textarea id="comments" name="comments" rows="5"></textarea>
<br>
<label for="country">کشور:</label>
<select id="country" name="country">
    <option value="iran">ایران</option>
    <option value="germany">آلمان</option>
</select>
<br>
<button type="submit">ثبت نام</button>
<fieldset>, <legend>

<fieldset>, <legend>

برای گروه‌بندی منطقی عناصر فرم و اضافه کردن عنوان به گروه.


<fieldset>
    <legend>اطلاعات تماس</legend>
    <label for="fullname">نام کامل:</label>
    <input type="text" id="fullname" name="fullname">
</fieldset>

تگ‌های مربوط به جداول (Tables)

برای نمایش داده‌ها به صورت ساختاریافته (ردیف و ستون).

<table>, <thead>, <tbody>, <tfoot>, <tr>, <th>, <td>

ساختار کامل جدول


<table>
    <caption>لیست محصولات</caption>
    <thead>
        <tr>
            <th>نام</th>
            <th>قیمت</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>لپ‌تاپ</td>
            <td>۱۵۰۰ دلار</td>
        </tr>
        <tr>
            <td>موس</td>
            <td>۲۵ دلار</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="2">مجموع ۲ محصول</td>
        </tr>
    </tfoot>
</table>

  • **<table>**: کانتینر اصلی جدول.
  • **<caption>**: عنوان جدول.
  • **<thead>**: سربرگ جدول.
  • **<tbody>**: بدنه جدول.
  • **<tfoot>**: پاورقی جدول.
  • **<tr>**: ردیف جدول.
  • **<th>**: سلول سربرگ (بولد).
  • **<td>**: سلول داده‌ای.
  • **colspan/rowspan**: برای ادغام سلول‌ها.

تگ‌های چندرسانه‌ای (Multimedia)

برای نمایش فایل‌های صوتی و تصویری.

<audio>

<audio>

برای قرار دادن فایل صوتی.


<audio controls>
    <source src="audio/music.mp3" type="audio/mpeg">
    <source src="audio/music.ogg" type="audio/ogg">
    مرورگر شما از عنصر audio پشتیبانی نمی‌کند.
</audio>
<video>

<video>

برای قرار دادن فایل ویدیویی.


<video width="640" height="360" controls poster="images/video-poster.jpg">
    <source src="video/my-video.mp4" type="video/mp4">
    <source src="video/my-video.webm" type="video/webm">
    مرورگر شما از عنصر video پشتیبانی نمی‌کند.
</video>
<iframe>

<iframe>

برای جاسازی (embed) یک سند HTML دیگر (مثل ویدیوهای یوتیوب).


<iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ"
        width="560" height="315"
        frameborder="0"
        allowfullscreen>
</iframe>

Entityهای HTML

کامنت‌ها

ویژگی‌های عمومی (Global Attributes)

ویژگی‌های عمومی (id, class, data-*, tabindex, title)

ویژگی‌هایی که در تقریباً هر تگ HTML قابل استفاده‌اند.


<div id="main-content" class="container" data-user-id="123" title="محتوای اصلی">
    <p>محتوای اصلی.</p>
</div>
<button tabindex="1">دکمه ۱</button>
